## 四、完整工作流

```
Step 1: 热点搜集（已有）
  └─ 热点雷达智能体 → 钉钉推送 → 结构化数据

Step 2: 数据整理（半自动）
  └─ 将推送内容提取为 JSON 格式（上方模板）

Step 3: 生成封面图（第1张）
  └─ GPT-Image-2 API → 1024x1536 竖版
  └─ 填入热点数据JSON → 全局总览

Step 4: 生成分平台详情（第2-5张）
  └─ 每平台一张 → 1024x1536 竖版
  └─ 抖音 / 小红书 / 微博 / 低分高赞

Step 5: 生成蹭点推荐（第6张·可选）
  └─ 1024x1536 竖版
  └─ 6个账号匹配建议

Step 6: 发布
  └─ 抖音图文 · 小红书图文 · 朋友圈九宫格
  └─ 按平台裁剪尺寸即可
```
